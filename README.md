# SUBIT-Business Analyzer

### Release v9.0 — Canonical Edition

A structural decision framework for evaluating business ideas using the **SUBIT-64 ontology**.

---

## Overview

**SUBIT-Business Analyzer** is a formalized strategic evaluation system based on the SUBIT model:

SUBIT = WHO × WHERE × WHEN
= 4 × 4 × 4
= 64 structural states

Each business idea is analyzed as a structured configuration inside a 6-bit semantic space:

WHO ∈ {ME, WE, YOU, THEY}
WHERE ∈ {EAST, SOUTH, WEST, NORTH}
WHEN ∈ {SPRING, SUMMER, AUTUMN, WINTER}

Which corresponds to:

{0,1}⁶
= {Anima, Animus}⁶
= 64 archetypal configurations

The system enables structural scoring, risk profiling, capital mapping, and monetization diagnostics.

---

## Core Purpose

The Analyzer is designed to:

* Formalize business intuition
* Detect structural asymmetry
* Identify capital efficiency zones
* Evaluate time-phase alignment
* Map monetization logic
* Produce structured go / pivot / reject decisions

---

## Structural Model

Every idea is evaluated across three orthogonal axes:

### 1. WHO — Agency Layer

Who captures value?

* ME — solo founder
* WE — team
* YOU — client-centered
* THEY — system/platform-driven

### 2. WHERE — Vector Layer

Where does value accumulate?

* EAST — innovation / technology
* SOUTH — distribution / market
* WEST — capital / ownership
* NORTH — authority / regulation

### 3. WHEN — Phase Layer

At what structural phase?

* SPRING — creation / experimentation
* SUMMER — growth / scaling
* AUTUMN — optimization / extraction
* WINTER — decline / transformation

---

## Output Format (Canonical)

Every analysis must produce:

1. SUBIT Code
2. Structural Interpretation
3. Risk Vector
4. Capital Efficiency Score (0–100)
5. Monetization Model
6. Structural Asymmetry
7. Go / Pivot / Reject Recommendation

---

## Scoring Logic

Capital Efficiency Score is derived from:

Founder Leverage × Market Vector × Phase Alignment − Structural Friction

Score ranges:

0–40 → Structural Misalignment
41–70 → Conditional Opportunity
71–85 → Strong Configuration
86–100 → Structural Advantage
